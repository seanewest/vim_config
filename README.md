A "Fairly decent" vim setup
=============
![Screen Shot](./screen_shot.png)

Install
=============
```
git clone --recursive https://github.com/seanewest/vim_config.git ~/.vim
ln -s ~/.vim/.vimrc ~/.vim/.gvimrc ~/
```

Plugins
=============

 * [ctrlp](https://github.com/kien/ctrlp.vim)
